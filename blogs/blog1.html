Password Manager Using C/C++ languages

Ok this blog will help you to create a password manager using c and c++ language. If anyone want to save your important password with encryption this project will help you a lot.

How actualy it's works

  • step1 : When you run this code you must need all your information that you want to save.
  • Note: The default password of this password manager is admin. If you want to change the password then you need to go the code's password function where you can change the master password.
  • step2: When you need to see the information you must enter the master password , if your entered password match with master password it will popup notepad with your information and when you task completed just close the notepad and your information will automacally encrypted and saved.
